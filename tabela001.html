hierarquia tabela simples

<table> tabela
<tr> table row, linha da tabela, cada <tr> e uma linha table header, cabeçalho da tabela
<td> table data dados da tabela
se a hierarquia não for obedecida a informação pode aparecer antes ou depois da tabela
obs:.para comentar algo dentro do codigo usamos <---->

Na seção <style> usamos 'table-width' para largura da tabela
'height'paraaltura
'border-collapse:' para determinar se as linhas da tabela ficarão
juntas(colapse) ou se havera espaço entre elas

Em 'td' podemos usar 'border: 0px solid cor' para determinar
espesura da borda, tipo de borda, cor da borda; 'text-align' para
posicionar o texto na horizontal e 'vertical-align:'para posicionar o terxto na vertical com os comandos 'midle = centro'; top=topo; botton= baixo; left=esquerda; hight=direita 'backgound-color' para cor de preenchimento,
se usarmos uma 'class' para uma determinada 'td' alteramos so essa 'td'
o mesmo serve para 'tr' e 'th'
